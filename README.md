# minecraft_ctl
Python script to manage minecraft server on docker on digitalocean droplet

# Usage
```
 git clone https://github.com/hungcat/minecraft_ctl
 cd minecraft_ctl
 pip3 install -r requirements.txt
 python3 mc_ctl.py help
```
